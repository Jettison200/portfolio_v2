Who Is Jason Clarke?

I am a Jamaican web developer with a passion for front end development and design. I Became a web developer because I noticed that my country had very few quality websites and fewer web developers. I aspire for a career that will allow me to channel my creativity through crafting beautiful software and engaging experiences.

I specialize in creating seamless user experiences with clean, efficient and scalable code. I consider web development to be an ongoing education and I am always looking for opportunities to work with those who are willing to share their knowledge as much as I want to learn. At the end of the day, my primary goal is to create something beautiful that has a positive impact on those who view it.

Expertise

WEB DEVELOPMENT

Using HTML, CSS and JavaScript along with additional libraries, I create beautiful websites with amazing functionalities. I have a passion for developing pixel-perfect websites while maintaining a semantic, modular and DRY code base.

RESPONSIVE UI DESIGN

I strive to develop and implement responsive and aesthetically pleasing interfaces. My websites adapt to any type of device, platform or browser.

EXPERIENCE DESIGN

User experience is an aspect of web development which I believe is vital to the success of a website. I believe that it is not only about what the website does. How the website looks and makes you feel must also be taken into consideration.

HOSTING AND SEO

Web hosting allows you to place your website onto the Internet. SEO allows a website to be found when by others using a search engine such as Google. I am very familiar with both.

Portfolio

My Recent Projects

Herbert Morrison Technical High School

While serving as the Senior Web Developer for the Engineering Club, I was tasked with creating a website for my School Herbert Morrison Technical High School. I designed and developed their current website using HTML, CSS and Javascript. I was responsible for the maintenance, hosting and SEO of the website.

Lawyers Of Jamaica.

Lawyers Of Jamaica is a demo web page for a Jamaican law firm. The website enables you to view various pages and aquire the necessary information that you would expect to find on a law firm's website. Information such as case studies, a list of all the lawyers who represent the firm and even a page dedicated to the law firm's practice areas.

Covid-Ja

This website was created to educate the Jamaican public on the Coronavirus. As there were no websites specifically for Jamaica and it's unique circumstance, I saw the need to create the first. Along with relevant information on the Coronavirus, the website also hosts an interactive map which gave live details of the status of the virus in all countries of the world.

Mount Alvernia High School

After completing the website for Herbert Morrison Technical High School, the website coordinator for the Mount Alvernia High School made contact and requested a website for their institution.

As a team, a local developer by the name of Suniel England and I designed and developed the website using HTML, CSS, Javascript along with a few Javascript libraries.

Robot
